# 🌍 TripNest – TripAdvisor Clone Website

TripNest is a responsive front-end travel website inspired by TripAdvisor.  
It is built using HTML and CSS to replicate a real-world travel platform UI including search, destinations, hotels, restaurants, and footer sections.

## 🎯 Project Objective

To replicate the official TripAdvisor website layout and design using only HTML and CSS.  
This project focuses on practicing UI structure, layout design, and responsive web development.

## 📱 Responsive Design

- Built using CSS Flexbox/Grid
- Mobile-friendly layout
- Works on desktop, tablet, and mobile screens

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Font Awesome Icons  

## 🚀 Learning Outcome

- HTML page structure and semantic layout  
- CSS Flexbox/Grid design  
- UI replication skills (TripAdvisor clone)  
- Responsive web design practice  
- Real-world website structuring
